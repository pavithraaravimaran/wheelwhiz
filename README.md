# ♿ WheelWhiz - Wheelchair Assistance App

WheelWhiz is a Flutter-based mobile application developed to improve accessibility and independent mobility for wheelchair users. The application assists users in navigating safely by providing information about wheelchair-friendly locations and essential services nearby.

The primary objective of the project is to make daily travel more inclusive, safe, and convenient by enabling users to quickly locate accessible facilities, receive navigation assistance, and access emergency support whenever required.

The application is built using **Flutter** for cross-platform mobile development and **Firebase** for user authentication and cloud data storage.

---

# 📌 Project Overview

People using wheelchairs often face challenges while traveling due to the lack of information about accessible locations and emergency support. WheelWhiz addresses these challenges by providing an easy-to-use mobile application that helps users discover nearby wheelchair-accessible services and navigate to them efficiently.

The application integrates location-based services with Firebase to deliver a seamless and responsive user experience while ensuring accessibility remains the primary focus.

# ✨ Features

### 🔐 User Authentication
- Secure user registration and login using Firebase Authentication.
- Personalized user access and data management.

### 📍 Accessible Navigation
- Provides navigation to wheelchair-friendly locations.
- Helps users travel independently with real-time location guidance.

### 🏥 Nearby Hospitals
- Displays nearby hospitals for quick medical assistance.
- Helps users reach healthcare facilities during emergencies.

### 🚻 Accessible Restrooms
- Locate nearby wheelchair-accessible restrooms.
- Improves convenience while traveling.

### 🔧 Wheelchair Repair Shops
- Finds nearby wheelchair repair and maintenance centers.
- Ensures users can quickly locate repair services when needed.

### 🆘 Emergency Assistance
- Provides quick access to emergency support.
- Designed to help users during critical situations.

### ☁️ Firebase Integration
- Firebase Authentication for secure login.
- Cloud Firestore for storing user data.
- Real-time database synchronization.
